## ðŸ” Why use `prevCount => prevCount + 1`?

React state updates are **asynchronous**, meaning:

```jsx

```

is risky when multiple updates happen quickly. Instead, use:

```jsx
setCount(prev => prev + 1); // âœ… React guarantees this gets latest value
```
